Nadiya’s Clothing
   Nadiya’s Clothing is a fashion website that showcases clothing collections for women, men, kids, and unisex. The site is built using HTML, CSS, and JavaScript.

Features
    Home page with brand introduction
    Collections with clothing name, size, and price
    About Us page
    Buy Now and Add to Cart buttons
    Clean and user-friendly layout

Technologies Used
  HTML
  CSS
  JavaScript
